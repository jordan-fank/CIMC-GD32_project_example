## 🛠️ 技术栈

### 硬件平台
- **主控芯片**: GD32F470 / STM32F429IGT6
- **内核**: ARM Cortex-M4 @ 180MHz
- **内存**: 1MB Flash + 256KB SRAM
- **外设**: ADC/DAC, USART, SPI, I2C, SDIO, GPIO

### 软件架构
- **底层驱动**: STM32 HAL库
- **操作系统**: 无操作系统 (裸机)
- **任务调度**: 协作式调度器
- **通信协议**: WebSocket, HTTP, UART
- **文件系统**: FatFS, LittleFS
- **UI框架**: WouoUI, U8G2

### Web技术
- **后端**: Node.js, Express, WebSocket
- **前端**: HTML5, CSS3, JavaScript
- **通信**: WebSocket实时通信
- **部署**: PM2, Nginx, Let's Encrypt

## 🔧 系统要求

### 开发环境
- **操作系统**: Windows 10/11, Linux, macOS
- **IDE**: Keil MDK-ARM v5.37+
- **编译器**: ARMCC v5.06+ / GCC ARM
- **调试器**: ST-Link V2/V3
- **Node.js**: v16+ (Web服务)

### 运行环境
- **硬件**: GD32F470 / STM32F429开发板
- **内存**: 最小64KB Flash, 20KB RAM
- **外设**: 基础GPIO, USART, 定时器
- **网络**: 4G模块 (可选)
- **存储**: SD卡 (可选)
